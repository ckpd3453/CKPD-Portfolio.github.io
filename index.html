My Tech-stack

Vanilla JavaScript

"Vanilla JS" is a term used in web development to refer to using plain JavaScript without the use of any additional libraries or frameworks. It essentially means working with the core JavaScript language and the native APIs provided by web browsers to build web applications

>

React, Bootstrap & Mui

"React JS," is an open-source JavaScript library developed by Facebook for building user interfaces. It is primarily used for developing single-page applications and dynamic web interfaces.

Material-UI is a popular open-source React UI framework that implements Google's Material Design guidelines.

Bootstrap is a popular open-source front-end framework developed by Twitter.

Node JS, Express

"Node JS" is an open-source, server-side runtime environment that allows developers to build and run JavaScript applications outside of a web browser. It is built on the Chrome V8 JavaScript engine and provides a way to execute JavaScript code on the server, enabling the development of scalable and high-performance network applications.

"Express" Express.js, commonly referred to as Express, is a minimal and flexible web application framework for Node.js.

MongoDB, MySql

"MongoDB" is a popular, open-source, NoSQL database management system that is designed to store and manage large volumes of unstructured or semi-structured data. MongoDB is known for its flexibility, scalability, and support for a wide range of use cases.

"MySQL" is an open-source relational database management system (RDBMS) that is widely used for managing and organizing structured data. It is known for its robustness, scalability, and reliability.

HTML, CSS

"HTML" is a markup language used to structure the content of web pages. It provides a set of elements and tags that define the structure and semantics of a document. HTML documents are interpreted by web browsers to render text, images, links, forms, and multimedia.

"CSS is a style sheet language used for describing the presentation and layout of web documents written in HTML. It allows web developers to control the visual appearance of web content, including colors, fonts, spacing, and positioning.

GitHub, Docker

"GitHub" is a web-based platform and service that provides a wide range of tools and features for version control, collaboration, and software development. It is a central hub for developers, allowing them to host and manage their code repositories, collaborate with others, and track changes to their projects.

"Docker" Docker is a platform and tool that enables developers to build, package, and distribute applications and their dependencies as lightweight containers. Containers are isolated environments that contain everything needed to run an application, including the code, runtime, system libraries, and settings.
